# Gems

```bash
gem install <gemname>
```

- bundler2.1.4 :Allows installation of gems specific to this project
  - run `bundler init` to create the bundler file in your folder
  - Gems are tied to your ruby version, on ruby upgrade gems will need to be re-sourced
  - Use `bundle add` to add specific gem files to the project
  - Alternatively, add the gem file line into the `Gemfile` created by `bundler init`

- Gems can be built using bundler. Look up the relevant documentation `gem bundler documentations`
